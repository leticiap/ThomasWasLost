# Thomas Was Lost
Based on the puzzle game Thomas Was Alone. Your objective is to take the two characters to the goal (white square) before the timer reaches 0. The game has four levels.
### Commands
- Move player 1: WASD
- Move player 2: arroy keys
- Change camera: E
